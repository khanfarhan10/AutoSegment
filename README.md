# AutoSegment
Perform Automated Image Segmentation (Unsupervised/Unsupervised/Supervised) for faster Annotations

Primary RGB based - https://www.immersivelimit.com/tutorials/create-coco-annotations-from-scratch  
Advanced - kMeans or other clustering algorithms  
PenUltimate - cv2 filters + clustering  
Ultimate - Shannon Entropy + Clustering
